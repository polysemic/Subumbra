# LibreChat

LibreChat is in scope for a direct, app-owned Subumbra integration.

Start here:

- [install.md](./install.md)

Current round scope:

- clean Docker install only
- one custom LibreChat endpoint backed by Subumbra
- one routed success proof
- one fail-closed proof for an invalid or unscoped key ID

Deferred:

- takeover / preinstall migration
- plugins, agents, MCP, tool store, OAuth, SSO
- multi-endpoint LibreChat setups
