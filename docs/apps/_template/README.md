# App Integration Template

Use this folder as the starting point for a new app-owned integration doc set.

This landing page should explain:

- what the app is
- what Subumbra path is currently proven
- which operator paths exist in this folder
- which round established the current proof

Start here:

- [install.md](./install.md)
- add `takeover.md` only if the app has a separately proven migration path

Required notes for future implementations:

- keep scope explicit
- separate proven behavior from deferred future work
- link to extracted operator-facing templates under `templates/` when present
