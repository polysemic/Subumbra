# Subumbra 1.1.1-alpha Release Notes

*Release date: 2026-05-20*

`1.1.1-alpha` is a focused security patch release on top of `1.1.0-alpha`.

Compare:

- `1.1.0-alpha...1.1.1-alpha`

---

## Highlights

### Worker policy binding is now server-authoritative

The Worker now ignores a client-supplied `policy_hash` when building the
decrypt-time Durable Object payload for V3 `/proxy` requests. Instead, it uses
the live registry `policy_hash` as the decrypt-time authority.

This keeps the V3 request shape compatible while closing the integrity gap
identified during Shannon-assisted review of the staging Worker path.

### Security review findings stayed narrow

Auth and authz review did not produce a confirmed unauthenticated compromise.
The main credible runtime issue was the `policy_hash` authority bug, which is
what this release patches.

---

## What Changed Since 1.1.0-alpha

### Changed

- Worker `/proxy` now sends only the live registry `policy_hash` as the
  decrypt-time `policyHash`
- release-facing docs now describe the server-authoritative `policy_hash`
  behavior accurately

### Security

- patched the Shannon-confirmed policy-binding integrity gap
- staging verification confirmed:
  - transparent proxy requests still succeed
  - direct Worker `/proxy` requests still succeed with valid auth
  - tampering the client `policy_hash` no longer controls decrypt-time behavior
  - invalid adapter tokens still return `401 unauthorized`
- published a sanitized public-facing summary at
  `security/reports/2026-05/shannon-r75-summary.md`

## Upgrade Notes

- Existing operators upgrading from `1.1.0-alpha` do not need manifest changes
  for this release.
- The client request shape still carries `policy_hash`, but the Worker now uses
  the live registry `policy_hash` as the decrypt-time authority.

---

## Known Notes

- This is still an **alpha** release.
- `/proxy` pre-auth throttling remains a deferred defense-in-depth item, not a
  release blocker in this patch.
- The SSRF scan lane was limited by Shannon harness/container reachability and
  did not produce a confirmed product finding in this release cycle.
